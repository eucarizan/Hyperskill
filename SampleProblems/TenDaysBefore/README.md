#  10 days before

## Description
Write a program that reads a date from the standard input and prints a date that is 10 days before.

## Examples
**Sample Input 1:**
```console
2017-06-03
```

**Sample Output 1:**
```console
2017-05-24
```

**Sample Input 2:**
```console
2017-04-28
```

**Sample Output 2:**
```console
2017-04-18
```
