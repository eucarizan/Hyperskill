# A stream without zero

## Description
You need to return a `LongStream` consisting of all numbers from `-n` to `n` inclusively, but skipping the `0` value.

## Examples
**Sample Input 1:**
```console
3
```

**Sample Output 1:**
```console
-3 -2 -1 1 2 3
```
