# Print elements of a stream

## Description
Implement a method that prints out each element from a given stream of numbers except the first two elements.

Please, use Java Stream API to solve the problem.

## Examples
**Sample Input 1:**
```console
1 1 3 5 7 9
```

**Sample Output 1:**
```console
3
5
7
9
```
