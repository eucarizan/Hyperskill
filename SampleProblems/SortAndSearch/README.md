# Sort and search

## Description
Sort the following array in descending order and write the index number of the value 9.

`{3,4,9,2,1,4,5,8,3,4,0,12}`

Please note that the index number of the first element is 0.

## Examples
**Sample Input 1:**
```console
{3,4,9,2,1,4,5,8,3,4,0,12}
```

**Sample Output 1:**
```console
1
```
