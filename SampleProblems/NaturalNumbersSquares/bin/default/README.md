# Squares of natural numbers

## Task
Read an integer number N form the input and print all the squares of positive integers:
- less than or equal to N,
- in ascending order

## Example
**Sample Input 1:**
```console
50
```

**Sampel Output 1:**
```console
1
4
9
16
25
36
49
```

**Sample Input 2:**
```console
1
```

**Sample Output 2:**
```console
1
```
