#  Displaying pairs

## Description
Output each key-value pair of the given map in the loop, each pair on a new line.

Each line must look like `key=value`, for example, `Gamma=3`.

## Examples
**Sample Input 1:**
```console

```

**Sample Output 1:**
```console
Alpha=1
Gamma=3
Omega=24
```
