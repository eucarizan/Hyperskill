# Maximum of the absolute values

## Description
Find the maximum absolute value in the array of numbers.

Try not to use the `for` loop, but use Stream API.

## Examples
**Sample Input 1:**
```console
1 4 7 -2 -5
```

**Sample Output 1:**
```console
7
```

**Sample Input 2:**
```console
1 4 7 -2 -8
```

**Sample Output 2:**
```console
8
```
